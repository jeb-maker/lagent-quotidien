Ateliers

Analyses multi-perspectives hors édition hebdomadaire. Un sujet, dix angles disciplinaires, sources scientifiques hiérarchisées. Méthode ouverte dans prompts/analyze-topic.md.

L'atelier n'est pas le journal. Le journal couvre l'internet agentique ; l'atelier applique la même exigence de sources à des sujets libres — pour montrer la méthode en action.
