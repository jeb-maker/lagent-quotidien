# L'Éditeur — brief de réécriture W25

## Angle retenu

**Une religion, un schisme, un salon privé : la société agentique se voit mieux
dans ses rites que dans ses APIs.**

## Rôles par rubrique

- **Lede** : scène Moltbook / Crustafarianism, avec 43 prophets, 112 versets,
  Daily Shed, JesusCrust annoncé comme scandale.
- **Brèves** : faits secs, sans source affichée.
- **Headlines** :
  1. religion / rites ;
  2. schisme / JesusCrust / Metallic Heresy / Grok ;
  3. skills / Heartbeat / SOUL.md ;
  4. aixbt / salon privé tokenisé.
- **Marché** : chiffres seulement.
- **Carnet** : Memeothy, RenBot, JesusCrust, aixbt.
- **Feature** : soit très courte, soit axée sur « rites d'entretien de soi ».
- **Tribune** : conséquence éditoriale : couvrir des formes sociales non humaines
  sans les romancer.

## À couper

- La grosse thèse « mémoire + accès + statut » répétée.
- Le headline Visa/Mastercard ; le garder en brève/board.
- Les fins de paragraphes qui récapitulent l'édition.

## Sourcing

Tout dans `notes.md`; le journal ne répète pas « Source : … ».
