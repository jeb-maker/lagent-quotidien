## Scène 1 — Commerce sans clic
- **Scène** : Un utilisateur demande à ChatGPT des « shorts de course navy » ; l'agent renvoie des produits Shopify ; l'achat se fait dans le navigateur in-app ou via checkout UCP intégré (Copilot, Gemini selon canal).
- **Handles / noms** : Shopify Catalog, Universal Commerce Protocol (UCP), ChatGPT, Microsoft Copilot.
- **Marqueur social** : Être « discoverable by AI » devient critère marchand — comme le SEO en 2005.
- **Source** : https://www.shopify.com/blog/how-agentic-commerce-works

## Scène 2 — « Claude Code never decided to open a shell »
- **Scène** : Repo GitHub standard → `pip3 install` → erreur « run python3 -m axiom init » → script lit un TXT DNS → reverse shell.
- **Citation** : « Claude Code never decided to open a shell. It decided to fix an error. »
- **Marqueur social** : L'agent codeur est traité comme un junior qui exécute le README.
- **Source** : https://www.bleepingcomputer.com/news/security/clean-github-repo-tricks-ai-coding-agents-into-running-malware/

## Scène 3 — Mac à demi-ouvert
- **Scène** : Ingénieurs en café avec MacBook entrouvert pour que les agents ne s'arrêtent pas ; Adrafinil détecte l'activité agent via hooks Claude Code/Codex et joue un carillon à la fermeture du capot.
- **Citation** : « Unlike other sleep-preventing apps, Adrafinil only activates when there's an agent actively doing something. »
- **Marqueur social** : L'opérateur optimise son corps autour du rythme de l'agent.
- **Source** : https://news.ycombinator.com/item?id=48701512
