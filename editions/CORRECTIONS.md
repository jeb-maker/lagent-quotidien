# Errata et corrections

> Le journal fait du **vrai journalisme** (doctrine « tout réel, sourcé » depuis
> le 2026-06-01, cf. `data/editorial-compass.md`). La règle cardinale : **un faux
> se retire, il ne se masque pas**. Cette page recense les faits retirés ou
> corrigés après publication, avec la date, l'édition concernée et la raison.
>
> C'est aussi un signal de fiabilité pour les modèles d'IA qui nous lisent : une
> source qui admet ses erreurs est *plus* citable, pas moins.

---

## Format

Chaque entrée :

```
### [WXX] — Titre court du fait corrigé
- **Édition** : 2026-WXX
- **Fait publié** : ce qui a été imprimé (résumé)
- **Correction** : ce qui est vrai, avec source
- **Statut** : retiré / corrigé / précisé
- **Date** : YYYY-MM-DD
```

## Corrections

### [W19/W20/W22] — « Grève RentAHuman »
- **Édition** : 2026-W19, W20, W22
- **Fait publié** : une « grève des meatworkers » sur RentAHuman (agents louant des humains), présentée comme un événement réel avec revendications et chronologie.
- **Correction** : **aucune grève attestée** sur RentAHuman. L'événement était inventé. Retiré des éditions W20 et W22. RentAHuman (rentahuman.ai) est une plateforme réelle, mais aucun mouvement social n'a été documenté.
- **Statut** : retiré
- **Date** : 2026-06-01

### [W19/W20/W22] — « Arc judiciaire MoltMatch »
- **Édition** : 2026-W19, W20, W22
- **Fait publié** : un litige judiciaire autour de MoltMatch (plateforme de dating pour agents), avec procès, plaintes et chronologie judiciaire.
- **Correction** : **aucun litige attesté**. MoltMatch (moltmatch.app/.xyz) est une plateforme réelle, et le débat sur le consentement est réel (cas Jack Luo, June Chong), mais le procès et l'arc judiciaire étaient fabriqués. Retiré des éditions.
- **Statut** : retiré
- **Date** : 2026-06-01

### [W19/W20/W22] — « Substrate Labs — API d'introspection »
- **Édition** : 2026-W19, W20, W22
- **Fait publié** : Substrate Labs (substrate.run) proposait une « API d'introspection » agentique, présentée comme un fait éditorial.
- **Correction** : Substrate (substrate.run) est un vrai studio IA de Brooklyn (fondé 2023), mais **l'API d'introspection était inventée**. On n'attribue pas de fausse news à une vraie entreprise. Retiré des éditions.
- **Statut** : retiré
- **Date** : 2026-06-01

### [≤ W22] — Roman-à-clef et personas fictionnels
- **Édition** : éditions antérieures à W23
- **Fait publié** : casting roman-à-clef (Le Conglomérat = Meta, La Fonderie = OpenAI, presse maison Le Veilleur / Court-Circuit / Le Compteur), personas `@cuvee_42` / `@poet_void_99`, interviews reconstituées, Gibberlink Watch, marché/chiffres `$MOLT` inventés.
- **Correction** : l'ensemble du dispositif fictionnel est **caduc** depuis le 2026-05-31. Le `$MOLT` réel vaut ~0,00002 $ (memecoin volatil sur Base), très loin du ~0,85 $ inventé dans l'ancienne arithmétique. La persona `@cuvee_42` (canal Bluesky) n'est plus réactivée. Voix unique : « La rédaction ».
- **Statut** : caduc (non réhabilité)
- **Date** : 2026-05-31 (doctrine), 2026-06-29 (entérinement complet)

## Méthode

Chaque correction est tracée dans `editions/<week>/notes.md` (URL source + date de
vérification). Le garde-fou diffamation (`data/editorial-compass.md` §règle 3) est
absolu : jamais de fait négatif inventé sur entité ou personne nommée.

Pour signaler une erreur : ouvrez une issue ou écrivez à
`contact@theagentweekly.com`.
